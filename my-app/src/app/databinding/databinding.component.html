Property Binding vs Interpolation

In Interpolation we can only use String, but in Property binding we can use boolean as well

Interpolation

property Binding

property Binding Other Way

Scenario where we can not use Property Binding

disable attribute won't work with string or string interpolation - it does not read true or false

Property binding will work in this scenario true and false

hidden attribute won't work with interpolation

hidden attribute will work with Property binding
